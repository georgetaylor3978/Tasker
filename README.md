# Tracker Keeper

Personal task, habit stack, and module tracker. Built as a Progressive Web App (PWA) — installable on Android via Chrome.

## Features
- Modules with stacks and tasks
- Recurring habits (daily, weekly, monthly, etc.)
- Start time scheduling and sort order
- Completion history tracking
- Offline capable

## Install on Android
1. Open the link in Chrome
2. Tap the menu → "Add to Home Screen"
